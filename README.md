# 🌊 Sardinien 2026

Unsere gemeinsame Reise-App für die drei Familien · 5.–18. September 2026 · Sardinien.

**Live:** https://miheinecke.github.io/sardinien-app/

## Anmelden

1. Öffne den Link im Browser.
2. Gib das Familien-Passwort ein.
3. Trag deinen Namen ein und wähl eine Farbe.
4. Wer dieselbe Farbe nimmt, gehört zur selben Familie und teilt sich eine persönliche Packliste.

## Auf den Home-Screen installieren

So sieht die App wie eine richtige App aus — mit Icon auf dem Startbildschirm.

### iPhone (Safari)
1. Link in Safari öffnen.
2. Teilen-Button unten (📤).
3. «Zum Home-Bildschirm» → «Hinzufügen».

### Android (Chrome)
1. Link in Chrome öffnen.
2. Menü oben rechts (⋮).
3. «App installieren» bzw. «Zum Startbildschirm hinzufügen».

## Was ist drin

- 🎒 Packliste — gemeinsame Liste + eine pro Familie
- 🛒 Einkaufsliste vor Ort
- ☀️ Wetterprognose & Tipps für September
- 🏖️ Ausflugsideen mit Karten-Links
- 🍝 Mahlzeitenplan mit Rotation
- 💶 Gemeinsame Kasse / Splitwise-light
- 🚑 Notfallnummern Italien
- 💡 Tipps für die Reise

Alles wird zwischen allen Geräten live synchronisiert.

## Technisch

Statische HTML-App mit Firebase Realtime Database für Sync. Keine Build-Tools, keine Abhängigkeiten. `index.html` öffnen reicht.
